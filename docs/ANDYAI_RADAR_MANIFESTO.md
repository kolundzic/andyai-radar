# AndyAI Radar Manifesto

AI is no longer just one model or one chatbot. The modern AI stack includes models, local runtimes, interfaces, RAG, agent logic, workflow automation, developer cockpits, action agents, governance and human authority.

AndyAI Radar exists to track those signals, classify them, evaluate risk, and turn them into governed production decisions.

## Core Line

Track the AI stack.  
Filter the hype.  
Build with governance.
