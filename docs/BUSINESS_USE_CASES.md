# Business Use Cases

| Use Case | Best Stack | Client Value |
|---|---|---|
| Private Knowledge Assistant | RAGFlow + Open WebUI/Ollama | Ask company documents and receive sourced answers |
| AI Workflow Automation | n8n + AI classifier | Forms, emails, leads, CRM and notifications under control |
| Local Private AI Lab | Ollama + Open WebUI | Private model and document testing |
| Governed Agent Prototype | LangGraph + approval nodes | Multi-step agent with human approval |
| Client AI App Prototype | Dify / Langflow | Fast MVP assistant or workflow app |
| Action-Agent Safety Review | OpenClaw-style review | Evaluate risky agents before use |
