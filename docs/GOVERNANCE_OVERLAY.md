# Governance Overlay

No AI tool becomes part of the AndyAI stack because it is popular.

It enters the roadmap only after:

- signal review
- stack classification
- business value test
- risk and permission review
- evidence check
- human approval

## Governance Questions

- What can this tool access?
- What can it change?
- What can go wrong?
- Who approves it?
- What evidence is logged?
- Can we stop it?
- Can we roll it back?
