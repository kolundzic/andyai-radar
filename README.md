# 🅰️ AndyAI Radar

**Signal Intelligence for the AI Stack**

AndyAI Radar is the signal intelligence layer of the **AndyAI.ai** ecosystem.

It tracks AI tools, GitHub repositories, models, workflows and market signals, then converts them into stack maps, risk analysis, integration priorities, business opportunities and governed production decisions.

## Core Rule

No AI tool becomes part of the AndyAI stack because it is popular.

It enters the roadmap only after it passes:

- signal review
- stack classification
- business value test
- risk and permission review
- evidence check
- human approval
